# 📊 Array Processing (C++)

This project is a **university laboratory work** focused on array processing using C++.
The program demonstrates working with arrays, algorithms, file handling, and basic data processing.

---

## 🚀 Features

* 🔢 Generate an array with random integers
* 📋 Display the array
* 📦 Split array into groups of *n* elements
* ➕ Calculate sum for each group and save to `e.txt`
* 🔼 Sort array in ascending order (ASC)
* 🔽 Sort array in descending order (DESC)
* 🧹 Remove duplicate elements (create new array `V2`)
* 📊 Calculate:

  * total sum of array
  * sum of unique elements
  * difference between sums
* 📁 Export results to `data.json`
* ➖ Show difference between arrays (`|V1 - V2|`)

---

## 🛠 Technologies

* C++
* STL (`vector`)
* File I/O (`fstream`)
* Basic algorithms (sorting, searching)
